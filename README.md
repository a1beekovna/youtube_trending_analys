# Прогнозирование популярности музыкальных видео YouTube  
**YouTube Trending Videos — Аналитический ML-проект**

Этот проект посвящён анализу трендовых музыкальных видео YouTube и построению модели, способной прогнозировать количество просмотров на основе метаданных — таких как категория, дата публикации, теги, лайки, страна и другие параметры.

## Структура проекта
```

project/
│── data/
│   └── youtube.csv
│
│── notebooks/
│   └── analysis.ipynb
│
│── src/
│   ├── data_cleaning.py
│   ├── feature_engineering.py
│   ├── model_training.py
│   └── visualization.py
│
│── README.md
│── requirements.txt

````

---

## Цели проекта
- Выполнить очистку датасета: дубликаты, пропуски, некорректные значения.  
- Проанализировать зависимости между характеристиками видео и просмотрами.  
- Построить регрессионную модель для прогнозирования популярности.  
- Выполнить визуализации по странам и категориям.  
- Оценить качество моделей с помощью **MAE** и **R²**.

---

## Используемый датасет
Датасет: **YouTube Trending Videos Dataset**

Он содержит данные о популярных роликах YouTube:  
- название  
- категория  
- количество просмотров  
- лайки, дизлайки  
- количество комментариев  
- страна  
- дата публикации  
- дату попадания в тренды  
- теги  

Данные были очищены и нормализованы перед анализом.

---

## Этап 1 — Очистка данных
Что сделано:
- удаление дубликатов  
- удаление аномальных значений  
- очистка тегов  
- логарифмирование признаков с большим диапазоном (`views`, `likes`)  
- нормализация числовых признаков  
- перевод дат в формат datetime  
- генерация новых признаков (день недели, час, категорийные признаки)

Код размещён в файле:  
`src/data_cleaning.py`

---

## Этап 2 — Исследовательский анализ данных (EDA)
Выполнено:
- распределения просмотров  
- boxplot для категорий  
- корреляционный анализ  
- анализ выбросов  
- сравнение просмотров между странами  

Визуализации доступны в ноутбуке:  
`notebooks/analysis.ipynb`

---

## Этап 3 — Обучение моделей
Модели, использованные в проекте:

- **Linear Regression**  
- **Random Forest Regressor**  
- **Gradient Boosting Regressor**  
- **XGBoost** *(при наличии)*  

Метрики оценки:
- **MAE — Mean Absolute Error**  
- **R² — коэффициент детерминации**

Лучшие результаты показала модель **Gradient Boosting**.

Код:  
`src/model_training.py`

---

## Этап 4 — Визуализация популярности
Графики по:
- странам  
- категориям  
- динамике просмотров  
- распределению популярных видео  
- влиянию признаков (feature importance)

Код:  
`src/visualization.py`

## Как запустить проект
### 1. Установить зависимости
```bash
pip install -r requirements.txt
````

### 2. Запустить Jupyter Notebook

```bash
jupyter notebook
```

### 3. Открыть ноутбук

```text
notebooks/analysis.ipynb
```

### 4. Проверить модели

```bash
python3 src/model_training.py
```
